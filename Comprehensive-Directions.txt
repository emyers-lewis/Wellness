https://docs.github.com/en/pages/getting-started-with-github-pages/creating-a-github-pages-site
(this tutorial includes pictures).


Once you have made a GitHub account: 
- Create a repository named WellnessClinic. It will remain public.
- Under <>Code click "uploading an existing file" under Quick Setup.
- Choose your files to upload. There MUST be an index.html file in order for your page 
  to work properly, this is the default page that will open to the site. Then "Commit" 
  your changes.
- To add folders: add file -> create new file. In Name Your File type in "folderName/" 
  then name the file whatever and now you can commit those changes. You can now open 
  that folder and upload files like normal. This is the ONLY way to upload/make a folder.
  	NOTE: The folder name must match the ones written in your code!
- If you have a subfolder inside of another subfolder type "firstFolderName/secondFolderName" then the file name. NOTE: If you make multiple subfolders this way, the folder in your root will say "firstFolderName/secondFolderName". Clicking will open into the second folder, to go to the first click the folder ".." (think of links and how "../file" works).
- After every file and folder is added, enter Settings -> Pages.
- Under Branch change None to Main, keep /(root), and save.
- Add a domain name and save.
- Click Visit Site. That url is how you visit the site!
